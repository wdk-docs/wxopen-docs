15. 审核撤回
==================================================================================

单个帐号每天审核撤回次数最多不超过1次，一个月不超过10次。

请求方式:

GET（请使用https协议）

https://api.weixin.qq.com/wxa/undocodeaudit?access_token=TOKEN
返回说明

{
"errcode":0,
"errmsg":"ok"
}
错误码说明

错误码	说明
0	成功
-1	系统错误
87013	撤回次数达到上限（每天一次，每个月10次）
