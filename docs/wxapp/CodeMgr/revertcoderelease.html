11. 版本回退¶

请求方式:GET（请使用https协议）

https://api.weixin.qq.com/wxa/revertcoderelease?access_token=TOKEN 返回说明

{ “errcode”:0, “errmsg”:”ok”, } 错误码说明

错误码 说明 0 成功 -1 系统错误 87011 现网已经在灰度发布，不能进行版本回退 87012 该版本不能回退，可能的原因：1:无上一个线上版用于回退 2:此版本为已回退版本，不能回退 3:此版本为回退功能上线之前的版本，不能回退
